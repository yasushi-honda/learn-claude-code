# Week 8: 上級・エンタープライズ

> **対応公式ドキュメント**: https://code.claude.com/docs/en/
> **想定所要時間**: 約7時間（7レッスン x 約60分）
> **難易度**: ★★★★☆ 〜 ★★★★★

## 週のテーマ

セキュリティ・サンドボックス・エンタープライズ展開・クラウドプロバイダー連携・組織管理設定・ベストプラクティス・データプライバシーなど、組織規模での安全な運用に必要な上級トピックを学びます。

## 学習目標

この週を終えると、以下のことができるようになります：

1. Claude Codeのセキュリティモデルとプロンプトインジェクション対策を理解し、適切なパーミッション設定ができる
2. OSレベルのサンドボックスを設定し、ファイルシステムとネットワークの分離を実現できる
3. エンタープライズ環境での導入オプション（Teams/Enterprise/Bedrock/Vertex/Foundry）を比較し、最適な選択ができる
4. Amazon Bedrock・Google Vertex AI・Microsoft Foundryの認証設定とLLMゲートウェイ構成ができる
5. サーバー管理設定とエンドポイント管理設定で組織全体のポリシーを一元管理できる
6. コンテキスト管理・CLAUDE.md設計・4フェーズワークフロー等のベストプラクティスを実践できる
7. データ使用ポリシーとZero Data Retentionを理解し、コンプライアンス要件に対応できる

## この週で身につくスキル

- セキュリティリスクの把握とプロンプトインジェクション多層防御の構築
- OSレベルのサンドボックスによる安全な自律実行環境の構築と設定カスタマイズ
- エンタープライズ環境への適切な導入方式の選択とプロキシ・ゲートウェイ設定
- クラウドプロバイダー（AWS/GCP/Azure）との認証・IAM/RBAC統合
- 組織ポリシーのサーバー配信・MDM配信と強制適用
- コンテキスト管理・CLAUDE.md設計・自動化パターンによる生産性最大化
- コンプライアンス要件（GDPR・SOC2・ZDR）への対応

## 前提知識

- Week 1〜7の内容（基本操作・設定・カスタマイズ・MCP・フック・CI/CD）
- クラウドサービス（AWS/GCP/Azure）の基礎知識
- 企業セキュリティポリシーの基本概念
- （推奨）IAM・認証・認可の基礎知識
- （推奨）MDM・デバイス管理の基本概念

## 推奨学習順序

Week 8のレッスンは番号順に進めることを推奨します：

1. **セキュリティ設計** と **サンドボックス環境** でClaude Codeの安全性の基盤を理解する
2. **エンタープライズ導入** と **クラウドプロバイダー連携** で組織への展開方法を学ぶ
3. **組織管理設定** でポリシーの一元管理を把握する
4. **ベストプラクティス** で効率的な運用パターンを習得する
5. **データプライバシー・ZDR** でコンプライアンス対応を完成させる
