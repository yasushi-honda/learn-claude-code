# Week 2: 基本操作・CLI

> 公式ドキュメント: [Interactive Mode](https://code.claude.com/docs/en/interactive-mode) / [CLI Reference](https://code.claude.com/docs/en/cli-reference)

## 学習目標

この週では以下を習得します：

1. 対話モードのキーボードショートカット・入力モード・@メンション・/コマンドを使いこなせる
2. CLI コマンドとフラグの体系を理解し、自動化スクリプトへの組み込みができる
3. コードベース探索・バグ修正・リファクタリング・テスト作成などの基本ワークフローを実行できる
4. Claude Code の内部ツール（Read / Edit / Write / Glob / Grep / Bash）の違いと使い分けを説明できる
5. Git との連携機能（コミット・ブランチ・PR 作成・コンフリクト解消）を活用できる

## 学習の進め方

1. まず本 README を読んで全体像を把握する
2. 01（対話モード）を読み、実際にショートカットを試してみる
3. 02（CLI）は辞書的に使えるように一通り確認する
4. 03（ワークフロー）は実際のプロジェクトで各ワークフローを試す
5. 04・05 は理解を深めたいときに参照する
