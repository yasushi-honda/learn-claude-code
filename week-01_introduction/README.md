# Week 1: 入門・セットアップ

> 公式ドキュメント: [Claude Code Overview](https://code.claude.com/docs/en/overview)

## 学習目標

この週では以下を習得します：

1. Claude Code が何であるかを理解し、できることの全体像を把握する
2. 自分の環境（macOS / Linux / Windows）に Claude Code をインストールできる
3. 初回セッションを起動し、プロジェクトへの基本的な質問と変更依頼ができる
4. エージェントループ（context → action → verify）の仕組みを説明できる
5. 複数のアカウント種別（Pro / Console / Bedrock 等）の違いを理解し、ログイン・ログアウトができる

3. インストール・認証（02 / 05）は実際に手を動かしながら進める
