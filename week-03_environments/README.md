# Week 3: 開発環境統合

> 対応する公式ドキュメント: [Settings](https://code.claude.com/docs/en/settings)

## 学習目標

この週では以下を理解・習得します：

- ターミナルを Claude Code に最適化する設定方法
- VS Code / Cursor 拡張機能のインストールと活用
- JetBrains IDE プラグインによる統合方法
- Claude Code デスクトップアプリの機能と使い方
- Web 版・モバイルアプリでのアクセス方法
- Remote Control によるローカルセッションのリモート操作

## 概要

Claude Code はターミナル上のCLIツールですが、様々な開発環境と統合することで生産性をさらに高めることができます。Week 3 では、日常的に使う開発環境（ターミナル、IDE、デスクトップアプリ、Web）それぞれに Claude Code を最適に組み込む方法を学びます。

どの環境から使っても、CLAUDE.md やメモリ設定など、コア機能は共有されます。自分のワークフローに合った環境を選んで使いこなしましょう。

## 前提知識

- Week 1・Week 2 の内容（基本的な Claude Code の操作）
- 普段使っている IDE やエディタの基本操作

## この週の進め方

1. 自分がよく使う環境（VS Code / JetBrains / ターミナル）のレッスンを優先的に読む
2. 実際に拡張機能やプラグインをインストールして試す
3. デスクトップアプリと Web 版も試して、用途に合わせて使い分けを理解する

## まとめ

- Claude Code は CLI だけでなく、VS Code・JetBrains・デスクトップアプリ・Web など多様な環境で使える
- どの環境でも CLAUDE.md・設定・会話履歴は共有される
- Remote Control でローカル環境をスマートフォンから操作することも可能
